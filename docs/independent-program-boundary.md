# Independent-program boundary

The Claude Code portfolio repository named in the implementation brief was a design-quality reference only. No file, fixture, prompt output, score, result, commit, or revision history was copied into this repository.

This program is independently scoped to Codex behavior:

- Codex `SKILL.md` progressive disclosure and explicit/implicit invocation;
- `agents/openai.yaml` invocation policy and UI metadata;
- Codex `/goal` lifecycle and completion-contract semantics;
- `codex exec --json` event capture;
- Codex version/model/effort controls;
- local-only raw trace handling for Codex session identifiers;
- separate reviewed-draft and Goal-activation workflows.

Similar portfolio qualities—predeclaration, fixtures, deterministic checks, failure preservation, CI, and reviewer documentation—are evaluation-engineering patterns, not transferred results or provider-neutral claims.
